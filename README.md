# NEN 2767 Decompositie Bibliotheek

Een gratis, open-source webapplicatie voor het opbouwen van hiërarchische decomposities volgens de NEN 2767 norm. Speciaal ontwikkeld voor waterschappen, maar bruikbaar voor alle beheerorganisaties.

## 🌐 Live demo
👉 **[Bekijk de app](https://JOUWGEBRUIKERSNAAM.github.io/nen2767-decompositie)**  
*(vervang JOUWGEBRUIKERSNAAM door jouw GitHub gebruikersnaam)*

---

## 📋 Wat kan de app?

- **Hiërarchische decompositie** opbouwen volgens NEN 2767:
  - Areaal Portefeuille → Complex → Beheerobject → Element → Bouwdeel → Gebrek
- **Volledige NEN 2767 bibliotheek** ingebouwd:
  - 62 Beheerobjecten
  - 110+ Elementen
  - 90+ Bouwdelen
  - 70+ Gebreken
- **Drag & drop** vanuit bibliotheek naar structuur
- **Meerdere Arealen** beheren
- **Exporteren** naar JSON (volledig herstelbaar) en CSV/Excel
- **Importeren** van eerder opgeslagen JSON
- Werkt **zonder installatie** — gewoon een HTML-bestand openen

## 🚀 Installatie op GitHub Pages

### Stap 1 — Repository aanmaken
1. Ga naar [github.com/new](https://github.com/new)
2. Naam: `nen2767-decompositie`
3. Zet op **Public**
4. Klik **Create repository**

### Stap 2 — Bestand uploaden
1. Klik op **Add file → Upload files**
2. Sleep `index.html` naar het uploadvenster
3. Scroll naar beneden, klik **Commit changes**

### Stap 3 — GitHub Pages inschakelen
1. Ga naar **Settings** (tandwiel, rechtsboven in je repository)
2. Klik links op **Pages**
3. Onder *Source*: kies **Deploy from a branch**
4. Branch: **main**, folder: **/ (root)**
5. Klik **Save**

### Stap 4 — Klaar!
Na ~1 minuut is de app live op:
```
https://JOUWGEBRUIKERSNAAM.github.io/nen2767-decompositie
```
